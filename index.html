Piskel

Create your own animation with Piskel!

  • Understandable interface
  • Many tools and possibilities
  • Export to PNG and JPEG

Tools

  • Pen tool
  • Line tool
  • Rectangle tool
  • Circle tool
  • Eraser tool
  • Paint bucket tool
  • Color picker tool

You can also choose a color from a color pallete and define the size of the paintbrush.

Frames

You can easily work with frames:

  • add frames,
  • copy frames,
  • delete frames,
  • change the order via drag'n'drop.

Keyboard shortcuts

Use keyboard to switch between tools and define your own shortcuts!

Size of your canvas

Change the size of your canvas:

  • 32 × 32
  • 64 × 64
  • 128 × 128

Enter the fullscreen mode to see your animation in work!
